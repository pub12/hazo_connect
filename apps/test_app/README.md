# hazo_connect test application

- run `npm install` at the project root to ensure dependencies are available
- execute `npm run build:lib` before testing to emit the `dist` output
- use `npm run demo:test` to run the sample integration script located in this directory
- update `config.ini` and `.env.local` with environment-specific details before making live requests
